## Requirements

 - Python 3.8


## Running tests
1- `source venv/bin/activate` where venv is your virtualenv directory

2- `python3 -m unittest discover`

OR

2- install Pytest for python 3 `sudo apt-get install python3-pytest`

3- run `pytest-3` (for a better interface)
